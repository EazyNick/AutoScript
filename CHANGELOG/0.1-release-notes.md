# AutoScript 0.1 릴리즈

AutoScript의 첫 번째 정식 릴리즈입니다. 시각적 워크플로우 편집기를 통해 노드 기반 자동화 스크립트를 생성하고 실행할 수 있는 완전한 자동화 플랫폼을 제공합니다.

## 주요 기능

### 🎨 시각적 워크플로우 편집기
- 드래그 앤 드롭으로 노드를 배치하고 연결
- 노드 간 데이터 전달과 실행 순서를 시각적으로 확인
- 자동 충돌 감지 및 방지

### 🔧 다양한 노드 타입
- 시작, 액션, 조건, 반복, 대기, 이미지, 엑셀 노드 등
- 엑셀 비교 노드: 단일 노드로 엑셀 파일 열기, 시트 선택, 비교, 닫기 작업 수행

### 📊 스크립트 관리 시스템
- 대시보드를 통한 스크립트 목록 및 통계 확인
- 스크립트별 활성화/비활성화 설정
- 실행 횟수, 실패 횟수 등 상세 통계

### 📦 패키징 및 공유
- 스크립트와 커스텀 노드를 ZIP 파일로 패키징
- 필요한 커스텀 노드 자동 감지 및 포함
- 다른 환경으로 쉽게 이동 및 공유

### 🛠️ 개발자 도구
- 노드 생성 스크립트: 템플릿 파일 자동 생성
- 노드 삭제 및 검증 도구
- 기능 구현 구역이 명확히 표시된 템플릿

## 시스템 요구사항

- **Python**: 3.10.6 이상
- **운영체제**: Windows 10/11
- **메모리**: 최소 4GB RAM 권장
- **디스크**: 최소 1GB 권장

## 설치 방법

1. **ZIP 파일 다운로드 및 압축 해제**
   - 이 릴리즈의 Assets에서 `AutoScript-0.1.zip` 파일을 다운로드
   - 원하는 위치에 압축 해제

2. **가상환경 생성 및 활성화**
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
   > **중요**: 가상환경 이름은 반드시 `venv`로 생성해야 합니다.

3. **패키지 설치**
   ```bash
   cd server
   pip install -r requirements.txt
   ```

4. **서버 실행**
   - 프로젝트 루트에서 `start-server.bat` 파일 실행
   - 브라우저에서 `http://127.0.0.1:8001` 접속

자세한 설치 가이드는 [INSTALLATION.md](INSTALLATION.md)를 참고하세요.

## 빠른 시작

1. 웹 UI에서 "노드 추가" 버튼 클릭 또는 `Ctrl+N`
2. 원하는 노드 타입 선택
3. 노드를 캔버스에 배치하고 연결
4. 각 노드의 설정 조정
5. `Ctrl+S`로 워크플로우 저장
6. `F5` 또는 `Ctrl+R`로 실행

## 키보드 단축키

- `Ctrl+N`: 노드 추가
- `Ctrl+S`: 워크플로우 저장
- `Ctrl+Z`: 노드 이동 되돌리기
- `F5` / `Ctrl+R`: 워크플로우 실행
- `Delete`: 선택한 노드 삭제
- `Esc`: 실행 중지
