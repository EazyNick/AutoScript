## [0.0.7] - 2026-01-07

### 설정 기능 개선

#### 스크립트 실행 간격 설정
- 기본 타임아웃 설정을 스크립트 실행 간격으로 변경
- 각 스크립트 실행 사이에 설정된 간격만큼 대기
- 0.1초 단위로 설정 가능 (0.1초 ~ 10초)
- 기본값: 0.5초
- 전체 실행 시 각 스크립트 사이에 간격 적용

#### 설정 모듈 구조 개선
- `server/settings/` 폴더 생성 및 설정 관련 모듈 통합
  - `execution_interval.py`: 스크립트 실행 간격 관리
  - `theme.py`: 테마 설정 관리
  - `language.py`: 언어 설정 관리
- 설정 관련 함수들을 중앙 집중식으로 관리

### 주요 변경사항

- `execution.defaultTimeout` → `execution.scriptInterval`
- 기존 타임아웃 개념에서 실행 간격 개념으로 변경

#### 실행 로직 개선

- `server/main.py`: 재시도 및 알림 설정 초기화 로직 제거

### 엑셀 비교 노드 개선

#### 엑셀 비교 노드 종합 노드화
- 엑셀 비교 노드를 종합 노드로 변경
- 파라미터로 원본/대상 엑셀 파일 경로를 직접 받아서 처리
- 내부에서 엑셀 열기, 시트 선택, 비교, 닫기를 모두 수행
- 별도의 엑셀 열기/닫기 노드 불필요

#### 파라미터 변경
- `source_execution_id`, `target_execution_id` 제거
- `source_file_path`, `target_file_path` 추가 (필수)
- `source_sheet_name`, `target_sheet_name` 추가 (선택, 기본값: "Sheet1")
- `visible`, `save_changes` 파라미터 추가

#### UI 개선
- 엑셀 비교 노드 파라미터에 파일 선택 버튼 추가
- PC 로컬 디렉토리에서 엑셀 파일 선택 가능
- "엑셀 열기 노드 필요" 경고 메시지 제거
