# AutoScript 사용자 가이드

AutoScript는 웹 기반 시각적 자동화 도구입니다. 노드를 드래그 앤 드롭하여 자동화 스크립트를 만들고 실행할 수 있습니다.

## 빠른 시작

> **설치 및 실행 방법은 [INSTALLATION.md](../INSTALLATION.md)를 참고하세요.**

### 1. 첫 번째 워크플로우 만들기

1. **시작 노드 추가**
   - 좌측 사이드바에서 "시작" 노드를 찾아 캔버스에 드래그
   - 또는 `Ctrl+N` 키를 눌러 노드 추가 모달 열기

2. **노드 추가 및 연결**
   - 원하는 노드 타입 선택 (클릭, 대기, 이미지 터치 등)
   - 노드를 캔버스에 배치
   - 노드의 출력 포트에서 다음 노드의 입력 포트로 드래그하여 연결

3. **노드 설정**
   - 노드를 더블클릭하거나 우측 상단 설정 버튼 클릭
   - 각 노드의 파라미터 설정 (좌표, 대기 시간, 이미지 경로 등)

4. **저장**
   - `Ctrl+S` 또는 저장 버튼 클릭
   - 워크플로우 이름 입력 후 저장

5. **실행**
   - `F5` 또는 `Ctrl+R` 또는 실행 버튼 클릭
   - 실행 중인 노드는 주황색, 완료된 노드는 초록색으로 표시

## 주요 기능

### 워크플로우 편집

- **노드 추가**: `Ctrl+N` 또는 "노드 추가" 버튼
- **노드 이동**: 노드를 드래그하여 위치 변경
- **노드 삭제**: 노드 선택 후 `Delete` 키
- **되돌리기**: `Ctrl+Z` (노드 이동/삭제 취소)
- **저장**: `Ctrl+S`
- **자동 배치**: 노드 크기 변경 시 다른 노드와 겹치지 않도록 자동으로 배치됨

### 워크플로우 실행

- **단일 실행**: 워크플로우 페이지에서 `F5` 또는 `Ctrl+R`
- **전체 실행**: 헤더의 "전체 실행" 버튼으로 활성화된 모든 스크립트 순차 실행
- **실행 중지**: `Esc` 키
- **실행 상태**:
  - 주황색: 실행 중
  - 초록색: 완료
  - 빨간색: 오류 발생

### 스크립트 관리

- **대시보드**: 전체 스크립트 목록 및 통계 확인
- **활성화/비활성화**: 대시보드에서 각 스크립트의 활성/비활성 버튼으로 실행 여부 제어
- **스크립트 로드**: 대시보드에서 스크립트 클릭하여 편집

### 설정

- **테마**: 라이트/다크 모드 전환
- **언어**: 한국어/영어 선택
- **실행 설정**: 스크립트 실행 간격 설정 (0.1초 단위)
- **스크린샷**: 자동 스크린샷, 오류 시 스크린샷, 저장 경로, 이미지 형식

## 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+N` | 노드 추가 |
| `Ctrl+S` | 워크플로우 저장 |
| `Ctrl+Z` | 되돌리기 (노드 이동/삭제) |
| `F5` / `Ctrl+R` | 워크플로우 실행 |
| `Delete` | 선택한 노드 삭제 |
| `Esc` | 실행 중지 |

## 노드 타입

### 기본 노드
- **시작**: 워크플로우의 시작점
- **종료**: 워크플로우의 종료점

### 입력 노드
- **클릭**: 지정한 좌표 클릭
- **이미지 터치**: 화면에서 이미지 찾아 클릭
- **키 입력**: 키보드 입력
- **마우스 이동**: 마우스 커서 이동

### 대기 노드
- **대기**: 지정한 시간만큼 대기
- **이미지 대기**: 특정 이미지가 나타날 때까지 대기

### 조건 노드
- **조건 분기**: 조건에 따라 다른 경로로 실행

### 로직 노드
- **반복**: 지정한 횟수만큼 반복 실행

### 엑셀 노드
- **엑셀 열기**: 엑셀 파일 열기
- **엑셀 시트 선택**: 특정 시트 선택
- **엑셀 읽기/쓰기**: 엑셀 데이터 읽기/쓰기
- **엑셀 닫기**: 엑셀 파일 닫기

자세한 노드 설명은 [노드 목록](node.md)을 참고하세요.

## 문제 해결

### 서버가 시작되지 않음
- Python이 설치되어 있는지 확인
- 포트 8001이 이미 사용 중인지 확인
- `.env` 파일에서 포트 번호 변경 가능
- 자세한 내용은 [INSTALLATION.md](../INSTALLATION.md)의 문제 해결 섹션을 참고하세요

### 노드가 실행되지 않음
- 노드 설정이 올바른지 확인
- 이전 노드가 정상적으로 완료되었는지 확인
- 실행 로그 확인 (개발자 도구 콘솔)

### 이미지를 찾을 수 없음
- 이미지 경로가 올바른지 확인
- 이미지 파일이 존재하는지 확인
- 화면 해상도와 이미지 크기 확인

### 워크플로우가 저장되지 않음
- 스크립트 이름이 입력되었는지 확인
- 네트워크 연결 확인
- 브라우저 콘솔에서 오류 메시지 확인
